## Quando usar
Para registrar peças de reposição com códigos, especificações técnicas e compatibilidade com equipamentos.

## Acesso
Menu lateral → Cadastros → Catálogo de Peças

## Visão geral da listagem
- Filtros por part number, códigos, família, marca.
- Ações: criar, editar, visualizar, excluir.

## Passo a passo — Nova peça (wizard 5 abas)
1) Clique em **Nova Peça**.  
2) **Aba Geral**: Parts Number*, Código Interno*, Código P / Fornecedor (opcionais), Família*, Fotos.
3) **Aba Técnico**: Marca, Especificações*, Unidade de Medida*, Quantidade Mínima, Quantidade Máxima, NCM.  
4) **Aba Descrições**: Descrição Interna*, Descrição para NF, Descrição do Fornecedor, Descrição Completa.  
5) **Aba Compatibilidade**: Equipamentos Compatíveis, Equipamentos Adaptáveis, Equipamento Principal.  
6) **Aba Resumo**: revise e clique **Cadastrar**.

## Campos e validações
**Geral**  
- Parts Number*: identificador único.
- Código Interno*: obrigatório; identificador interno da peça.
- Código P / Código Fornecedor: opcionais; identificadores complementares.
- Família*: seleção de família de peças.  
- Fotos: upload opcional.

**Técnico**  
- Marca: seleção de marca.  
- Especificações*: texto obrigatório.  
- Unidade de Medida*: texto (ex.: un, pc, m, kg).  
- Quantidade Mínima: numérico >0 quando informado.  
- Quantidade Máxima: numérico >0 e ≥ mínima quando informado.  
- NCM: seleção com busca.

**Descrições**  
- Descrição Interna*: obrigatória.  
- Descrição para NF: opcional.  
- Descrição do Fornecedor: opcional.  
- Descrição Completa: opcional (texto longo).

**Compatibilidade**  
- Equipamentos Compatíveis: múltipla seleção (originais).  
- Equipamentos Adaptáveis: múltipla seleção (opcional).  
- Equipamento Principal: seleção (opcional).

## Mensagens frequentes
- “Part number obrigatório”  
- “Familia obrigatória”  
- “Especificações obrigatórias”  
- “Unidade de medida obrigatória”  
- “Código Interno é obrigatório”
- “Quantidade mínima deve ser maior que 0” / “Quantidade máxima não pode ser inferior à mínima”  
- “Descrição Interna obrigatória”

## Erros comuns
- Não preencher o Código Interno.
- Informar Quantidade Máxima menor que a Mínima.  
- Omitir especificações técnicas que impactam compatibilidade.
